# 📊 SaaS Sheet App

Realtime collaborative spreadsheet web app (Google Sheets clone)

## 🚀 Features
- 🔄 Real-time collaboration
- 🧠 Excel-like formulas (SUM, IF, AVG)
- 📄 Multi-sheet tabs
- 🟢 Live cursors & presence
- 🔒 Cell locking system

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Realtime: Socket.io
- Database: MongoDB

## 📦 Installation

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
